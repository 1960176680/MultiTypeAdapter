#特性
- 轻盈、整个类库只有9个文件
- 全面、支持 bean type 之间 一对一 和 一对多 的关系绑定
- 职责单一、只负责本分工作，专注多类型的列表视图 类型分发，不会影响 view 的内容或行为
- 内存、没有性能损失，内存友好
- 可读、代码清晰干净
